# Cuaderno-Programaci-n
Práctica 3: Herencia y Módulos
**Archivos:** `dispositivos.py` (Módulo), `main.py`, `interfaz_led.py`

Introducción a la **Programación Orientada a Objetos (POO)** y la organización de código en múltiples archivos.

* **Módulos (Importación):** Se aprendió a separar el código en archivos distintos (`import dispositivos`) para no tener todo en un solo script gigante.
* **Herencia (POO):**
    * `Clase Dispositivo`: Clase padre que define atributos comunes.
    * `Clase Led` y `Clase Boton`: Clases hijas que heredan de Dispositivo y añaden funcionalidades específicas.
* **Interfaz:** Menú interactivo en consola para probar la importación de clases y métodos.
