# Cuaderno-Programaci-n
Práctica 4: Arquitectura MVC (Modelo-Vista-Controlador)
**Carpetas:** `modelo/`, `vista/`, `controlador/`

Implementación de un patrón de diseño profesional para separar la lógica, los datos y la interfaz visual.

* **Modelo:** Define la lógica de los sensores (incluyendo un sensor DHT11 simulado) y actuadores.
* **Vista:** Muestra mensajes formateados en la consola.
* **Controlador:** Gestiona la comunicación entre el hardware (modelo) y el usuario (vista).
