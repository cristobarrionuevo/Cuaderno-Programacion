# Cuaderno-Programaci-n
Práctica 5: Control Remoto IoT con Telegram
**Archivos:** `modelo.py`, `vista.py`, `main.py`

Integración de servicios en la nube utilizando la API de Telegram.

* **Vista (`TelegramView`):** Reemplaza la consola de texto por un chat real de Telegram.
* **Funcionalidad:** El usuario puede enviar comandos ("on", "off") desde su celular para controlar el LED en la Raspberry Pi desde cualquier lugar.
